Questions

  1. What does the server in the client server architecture do?
    • Server is the software that runs on a cumputer. This software answers the requests from the client. The website files is stored on the system, and the serwer provides these to the client.
  2. What does the client do?
    • This is the software (usually a browser) that runs on users computers. This takes information from the server and displays it in a humanfriendly way.
  3. What is an IP address?
    • IP adress is the computers way of knowing where a system is located on the internet.
  4. What is DNS?
    • DNS or Domain Name Server is sort of a phonebook for computers. It takes in the url and tells the computer the IP of the website we are looking for.
  5. What is the difference between the Internet and WWW?
    • The internet is the system that connects the networks together. World wide web is a part of that system, this system provides the modern websites we use today.
  6. Explain what HTML is
